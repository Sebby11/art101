Lab 4: Pseudocoding & Problem solving

Challenge

In this lab we were tasked with thinking up an everyday task and a program we would like to create then breaking these problems into a set of steps. We also had to create the correct file structure.

Problems

The biggest problems were coming up with the specific tasks, but in reality it wasn't a very big problem. Overall the lab went smoothly.

Results

Everyday Task Pseudocode

Task: Cooking Pasta

1. Gather necessary ingredients
2. Gather necessary cookware (pots, pans, plates, etc)
3. Set water to boil
4. Cook pasta
5. Drain water & dry pasta
6. Add sauce, cheese, garlic clove, etc.
7. Place silverware, plates, cups, etc.
8. Serve food
9. Eat food
10. Wash dishes
11. Clean kitchen

Computer Task Pseudocode

Task: Animated Game Character Movement

1. Illustrate character
2. Draw key frames
3. Draw in-between frames
4. Repeat 2-3 for different animations for walking, idle, & jumping
5. Program if standing still loop idle frames
6. Program if walking loop walking frames
7. Program if jump loop jumping frames
