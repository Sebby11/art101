Lab 13: Loops

Challenge

In this lab we were tasked with creating a FizzBuzz function that will output specific string values depending on number multiplicity. Numbers ranged from 1 to 200.

Problems

First, we needed to figure out a way to output the strings and we ended up using regular paragraphs within columns of a div. Next we had trouble aligning the paragraph strings to the div columns, which we fixed by not aligning text within the output div. Finally we had some trouble on string formatting, but not too much.
